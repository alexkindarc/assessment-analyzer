## Daily Use

### Analyze a Report
1. Go to app URL
2. Enter team password
3. Select report type (Results / Improvement / Plan)
4. Upload file (PDF or Word)
5. Click "Analyze Report"
6. Review and edit metadata
7. Save to Excel Online

### Report Types

| Type | When to Use | Analysis Focus |
|------|-------------|----------------|
| **Results Report** | End-of-cycle reports with data | Full analysis + metadata storage |
| **Improvement Report** | Documents what was done | Clarity of actions + connection to prior proposals |
| **Next Cycle Plan** | Forward-looking plans | Outcome/method quality, no results analysis |

## Admin Tasks

### To edit how reports are analyzed:
1. Log in with Admin password
2. Go to Configuration tab
3. Edit Rubric Guidance or Prompt templates
4. Changes apply to future analyses

### To add a unit name alias:
1. Configuration → Unit Registry
2. Find the unit
3. Add old name to "previous_names" column (semicolon-separated)

### To upload custom rubric:
1. Configuration → Custom Rubric
2. Upload PDF/Word or paste text
3. Custom rubric supplements default criteria
